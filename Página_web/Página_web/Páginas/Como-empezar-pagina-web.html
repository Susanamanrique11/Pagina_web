¿Cómo empezar a crear una página web?

Al hablar acerca de las páginas web, comenzamos a pensar en qué tan difícil puede ser crear una, por qué medios puedo crear una, cuánto me tardaré y demás, los colores, el contenido, TODO!

ㅤ

Pero no te preocupes, vamos por partes, inicialmente, vamos a responder unas preguntas...

ㅤ¿Qué tan difícil puede ser crear una página web?

Iniciemos comentando que hay 2 maneras de crear una página web; de una manera práctica y sencilla por medio de páginas web que ofrecen el servicio de crear tu propia página web de manera gratuita (con opciones limitadas), o de pago (con mayor cantidad de opciones), se trabaja a manera de plantilla, agregando fuentes, cuadros, similar a un Power Point o Powtoon donde vas agregando cuadros de texto, imágenes y demás, siendo las más comunes, wix, páginas de google, blogspot, entre otras.

ㅤ

Sin embago, el trabajar netamente desde una platilla nos dificulta muchas cosas, más que todo de funcionalidades y personalización, nos limitamos bastante... Y es ahí donde entra la segunda manera, una un poco más complicada, pero que sin duda vale la pena, y es de lo que hablaremos el día de hoy.

ㅤ¿Qué conocimientos necesito para crear una página web?

A pesar de que crear una página web suena terrorífico por todas esas abreviaturas, los p, h1, class, entre otros, puede ser realmente sencillo y divertido con las herramientas adecuadas, así, los conocimientos básicos que necesitas para empezar a crear una página web, son lo básico del lenguaje Html 5 y CSS, siendo el Html 5, el lenguaje en el que codificamos todo el contenido que contendrá nuestra página, y el lenguaje CSS, aquel que nos permite darle color, vida y estilo a nuestra página...

ㅤ

Como te decía, esto es netamente para comenzar, para realizar lo básico como algunos links, el contenido, una que otra imagen, pero si de verdad quieres adentrarte en este mundo de las páginas web, aprovechar todas sus ventajas y convertir tu página web en una de las mejores, comenzarás a aprender Javascript, un lenguaje que te permite hacer maravillosas creaciones, tales como este Carrousel que nos muestra la línea de tiempo de las páginas web, que permite recopilación de información mediante formularios, la creación de carritos de compras, entre otros.

ㅤ

ㅤ¿Cómo crear mi propia página web y cuánto me tardaré?

Inicialmente, el tiempo es muy relativo, depende de tu capacidad de aprendizaje, qué tan rápido aprendas, de cuán detallista y perfeccionista seas, entre otros.
